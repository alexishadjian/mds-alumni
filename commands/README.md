# Commandes MDS Alumni

Commandes pour guider l'agent IA dans le développement de la plateforme.

## Commandes disponibles

| Commande | Description |
|----------|-------------|
| `explore-and-plan.md` | Workflow EPCT générique (Explore, Plan, Code, Test) pour toute tâche |
| `create-prp.md` | Créer un PRP pour une nouvelle feature |
| `implement-prp.md` | Implémenter une feature en suivant un PRP existant |

## PRPs MVP

| PRP | Fichier | Statut |
|-----|---------|--------|
| Auth & Onboarding | `PRPs/prp-auth-onboarding.md` | À faire |
| Profils utilisateurs | `PRPs/prp-profiles.md` | À faire |
| Annuaire | `PRPs/prp-directory.md` | À faire |
| Back-Office Admin | `PRPs/prp-admin.md` | À faire |

## Usage

- **Créer un PRP** : Utiliser `create-prp.md` avec une description de feature
- **Implémenter** : Utiliser `implement-prp.md` avec le nom du PRP (ex: `prp-profiles`)
- **Tâche ad-hoc** : Utiliser `explore-and-plan.md` pour toute autre tâche
